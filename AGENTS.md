# AGENTS.md

This repo is a Phase 6 local-only scaffold for public-safe engineering deliverable templates.

## Operating Rules

- Keep all content synthetic and public-safe.
- Use `Status: scaffolded`, `Publication state: private/not-public`, and `Release state: not released`.
- Do not imply stamped engineering, certification, code compliance, legal approval, customer deliverables, safety approval, production readiness, released models, released datasets, Space releases, active service offerings, pricing, or turnaround promises.
- Do not include customer data, Foundation-private data, secrets, credentials, private URLs, production procedures, production BOMs, production CAD, production schematics, sealed source, private model artifacts, private logs, active client deliverables, or internal company product names.

## Validation

Run `scripts/validate-public-boundary.sh`, the plan-listed `rg` boundary/status scan, `git diff --check`, and `git status --short` before review.
